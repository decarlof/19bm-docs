==========
Procedures
==========

Structured, cora-consumable recipes for operations on 19-BM-FACT
hardware. Same conventions as the 2-BM ``procedures/`` section —
each page documents one procedure end-to-end (preconditions, typed
parameters, atomic steps, postconditions, failure modes) so that
every page maps cleanly onto a cora ``Method`` definition.

For the underlying hardware (component reference, PV names where
known), see :doc:`manual`.
